# 기능
## 밤과 낮
게임에는 밤과 낮이 있다. 밤에는 인간의 시야가 좁이지고 괴물의 시야가 넓어진다. 낮은 반대이다. 또한 밤낮의 여부에 따라 탈출 불가능한 루트도 있다.
